# TailWind_CSS

## Concepts Discussed

1. **CDN** : How to use Tailwind CDN
2. **Colors**: How to use Tailwind Colors
3. **Container Spacing**: Learn about Margins, Padding etc
4. **Typography**: Font & its font family related concepts here
5. **Sizing**: Width, Height, Min & Max Width Heights
6. **Position & Display**: Postion - Absolute, Relative Fixed, Sticky **|** Display: Inline, Inline-block, Flex
7. **Background & Shadows**: More about Backgrounds & Shadows (Box & text shadows)
8. **Borders**: Border width, radius & style
9. **Filters**: Contrast, Saturate, Brightness, Hue, Blur etc
10. **Breakpoints**: MediaQueries
11. **Flexbox**: More about Flexbox
12. **Grid**: More about Grid
13. **Transformation, Transition, Animation**: More about Transform, Transition & Animation
14. **Tailwind CLI**: How to use Tailwind CSS via CLI instead of using CDN
15. **Projects**: Build projects using Tailwind CSS
